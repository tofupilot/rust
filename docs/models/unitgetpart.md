# UnitGetPart

Part information with revision details for this unit. Every unit must have a part and revision.

## Fields

| Field | Type | Required | Description |
| --- | --- | --- | --- |
| `id` | `String` | :heavy_check_mark: | Part ID. |
| `number` | `String` | :heavy_check_mark: | Part number. |
| `name` | `String` | :heavy_check_mark: | Part name. |
| `revision` | `UnitGetRevision` | :heavy_check_mark: | Revision information for this unit. |
