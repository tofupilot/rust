# UnitListRevision

Revision information for this part. Every unit must have a specific revision.

## Fields

| Field | Type | Required | Description |
| --- | --- | --- | --- |
| `id` | `String` | :heavy_check_mark: | Unique identifier for the revision. |
| `number` | `String` | :heavy_check_mark: | Revision number (e.g., version) of the part. |
